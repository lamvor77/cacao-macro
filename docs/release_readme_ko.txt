카카오톡 자동화 — 사용 안내
=================================

이 문서는 컴퓨터를 잘 모르는 분도 이해할 수 있도록 쉽게 썼습니다.


1. 실행 방법
------------

cacao_macro.exe 파일을 더블클릭하면 됩니다. 카카오톡을 먼저 실행해
두시면 좋습니다.


2. 필요한 세 파일
------------------

프로그램이 정상적으로 켜지려면 아래 세 파일이 "같은 폴더"에 있어야
합니다.

  1) cacao_macro.exe        (프로그램 본체)
  2) .env                   (설정 파일)
  3) license_build.json     (사용 기간이 적힌 라이선스 파일)

세 파일 중 하나라도 없거나 다른 폴더에 있으면 프로그램이 켜지지
않거나 오류 메시지가 뜹니다.


3. 라이선스 오류 대응
----------------------

프로그램을 실행했는데 "라이선스" 관련 오류 창이 뜬다면:

  - "라이선스 파일이 없습니다"
      → license_build.json 파일이 cacao_macro.exe와 같은 폴더에
        있는지 확인하세요.

  - "라이선스가 만료되었습니다"
      → 사용 기간이 끝났습니다. 배포 담당자에게 새 license_build.json
        파일을 요청해 받은 뒤, 기존 파일을 교체하세요(프로그램을 다시
        설치할 필요는 없습니다).

  - "라이선스 검증 설정이 누락되었습니다"
      → .env 파일이 없거나 내용이 잘못되었습니다. 배포 담당자에게
        문의하세요.

  이 세 가지 외의 오류가 계속되면, 아래 "6. 문의 시 진단정보 전달
  방법"을 참고해 문의해 주세요.


4. 로그인 방법
--------------

프로그램 오른쪽 위에 있는 "Google로 로그인" 버튼을 누르면 인터넷
브라우저가 열립니다. 평소 쓰시는 구글 계정으로 로그인하면 됩니다.
로그인하지 않아도 프로그램은 정상적으로 쓸 수 있습니다(다만 여러
컴퓨터 사이의 자동 동기화는 로그인해야 켜집니다).

로그인 후 "승인 대기"라고 표시된다면, 관리자가 사용 승인을 해줄
때까지 기다려 주세요.


5. 데이터 백업
----------------

프로그램 오른쪽 위 "진단정보" 버튼을 누른 뒤 "백업 관리..." 버튼을
누르면 저장한 단톡방 목록/메시지를 백업할 수 있습니다.

  - "지금 백업" 버튼: 그 자리에서 바로 백업을 만듭니다.
  - 프로그램은 하루에 한 번, 종료할 때 자동으로도 백업을 만듭니다.
  - 예전 백업으로 되돌리는 "복구" 기능은 관리자만 사용할 수 있습니다
    (되돌리기 전 현재 데이터도 자동으로 안전하게 백업됩니다).


6. 문의 시 진단정보 전달 방법
------------------------------

프로그램에 문제가 있어 문의하실 때는, 어떤 상황인지 글로 설명하는
것보다 "진단정보"를 함께 보내주시면 훨씬 빠르게 도와드릴 수 있습니다.

  1) 프로그램 오른쪽 위 "진단정보" 버튼을 누릅니다.
  2) "진단정보 복사" 버튼을 누릅니다.
  3) 문의하시는 곳(카카오톡, 이메일 등)에 붙여넣기(Ctrl+V) 하시면
     됩니다.

이 진단정보에는 비밀번호나 로그인 정보 같은 것은 절대 포함되지
않으니 안심하고 보내셔도 됩니다.
