# 카카오톡 단톡방 자동 메시지 전송 프로그램

Version: 1.0

---

# 프로젝트 개요

본 프로젝트는 Windows 환경에서 실행되는 카카오톡 PC버전 자동 메시지 전송 프로그램이다.

사용자가 선택한 여러 단톡방에 정해진 메시지를 자동으로 반복 발송하는 업무용 프로그램을 개발한다.

---

# 목표

반복적인 카카오톡 홍보 및 안내 업무를 자동화한다.

프로그램은 안정성을 최우선으로 하며 실제 업무에서 매일 사용할 수 있는 수준으로 개발한다.

# 프로그램 구성

프로그램은 아래 기능으로 구성한다.

## 1. 활성화된 카톡방 관리

### 기능

현재 카카오톡에서 열려있는 단톡방 목록을 가져온다.

### 버튼

* 활성화된 카톡방 가져오기
* 목록 저장
* 저장된 목록 불러오기
* 저장된 카톡방 켜기

---

## 2. 활성화된 카톡방 목록

왼쪽에는 단톡방 목록 표시 영역이 있다.

기본 상태에서는 비어 있다.

다음 경우에만 표시한다.

* 활성화된 카톡방 가져오기
* 저장된 목록 불러오기

표시 형식

□ 간병정보1

□ 간병정보2

□ 간병정보3

...

체크된 단톡방만 발송 대상이다.

---

## 3. 저장 기능

JSON 저장

저장 내용

* 단톡방 이름
* 체크 상태
* 메시지 1~12

프로그램 실행 시 자동 복원한다.

---

## 4. 저장된 카톡방 자동 열기

버튼

저장된 카톡방 켜기

동작

1. 카카오톡 실행 확인
2. 저장된 단톡방 검색
3. 채팅방 열기
4. 다음 단톡방 반복

이미 열린 채팅방은 다시 열지 않는다.

# 완료 기준

다음 조건을 모두 만족하면 프로젝트 완료로 본다.

* 단톡방 목록을 저장할 수 있다.
* 저장된 단톡방을 자동으로 다시 열 수 있다.
* 메시지 12개를 저장할 수 있다.
* 정각, 15분, 30분, 45분 자동 발송이 정상 동작한다.
* 단톡방 기준으로 메시지가 순차 발송된다.
* 메시지 간 2초 대기가 적용된다.
* 단톡방 이동 시 랜덤 딜레이가 적용된다.
* 로그가 정상 출력된다.
* EXE 파일에서 정상 동작한다.
