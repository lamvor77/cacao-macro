# Changelog

## v1.2.1 - 2026-07-22

### Fixed
- 메인 창 제목에 표시되던 버전 번호를 올바른 v1.2.1로 수정했습니다(이전
  배포본 내부에는 예전 버전 번호가 남아 있었습니다).
- 진단정보, 로그 시작 메시지, 백업 메타데이터, 빌드 스크립트가 모두
  단일 버전 소스(`config/version.py`)를 통해 동일한 버전을 표시하도록
  정리했습니다.
- TEST ENVIRONMENT 경고 표시 동작은 그대로 유지했습니다(운영 `.env`에서는
  표시되지 않고, 테스트 `.env`에서만 표시됩니다).

### Added
- 창 제목/버전 문자열 회귀를 막는 자동 테스트를 추가했습니다.

### Validation
- Python 테스트: 547개 통과
- 모바일 테스트: 39개 통과
- Legacy REST polling 재발 없음(2분 이상 관찰)
- Realtime 동기화 정상 동작 확인

## v1.2.0 - 2026-07-22

### Added
- Legacy 메시지 동기화를 이벤트 기반 방식으로 전환했습니다.
- Shared/Legacy 메시지 발송 직전 최신 상태 확인을 추가했습니다.
- Shared/Legacy 메시지 수동 새로고침 기능을 추가했습니다.
- 단톡방 목록을 저장할 때 저장 위치와 파일명을 직접 선택하는 네이티브
  저장 대화상자를 추가했습니다.

### Changed
- 모바일 메시지 등록 화면에서 제목 입력 기능을 제거했습니다.
- 프로그램 사용기간(만료일) 제한을 제거했습니다 — 사용 권한은 승인된
  계정 인증 방식으로 관리합니다.
- 배포 파일(exe)의 서명 검증은 그대로 유지했습니다(관리자가 발급한
  빌드인지 확인).
- Realtime 연결 프로토콜 호환성을 개선했습니다.

### Removed
- 30초 주기의 REST 폴링을 제거했습니다.
